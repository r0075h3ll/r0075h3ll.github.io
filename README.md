# Hardik Nanda Portfolio

Personal portfolio site for Hardik Nanda (`r0075h3ll`) focused on Product Security, DevSecOps, and vulnerability research.

## Run Locally

Prerequisite: Node.js

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Build

1. Create production build: `npm run build`
2. Preview production build: `npm run preview`

## Deploy

Publish to GitHub Pages: `npm run deploy`
